# Catchly

Privacy-first subscription tracker for Chrome. Local-only — no bank login, no data sold.

- **Source:** https://github.com/Muzeeb1998/Catchly
- **License:** MIT
- **Current version:** 0.1.0

## Pages

- [Privacy Policy](./privacy-policy.html)

## Install

Chrome Web Store listing pending. For now, clone the repo and load it as an unpacked extension:

```bash
git clone https://github.com/Muzeeb1998/Catchly.git
```

Then `chrome://extensions/` → toggle Developer mode → Load unpacked → select the cloned folder.
